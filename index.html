# 介绍

boat-ui是一个基于Vue开发的UI框架，目前提供了button-按钮、input-输入框、layout-布局、 tabs-标签、grid-网格系统、toast-弹窗、popover-气泡卡片、collapse-手风琴组件， 更多组件尚在开发中。

# 开发目的

boat-ui以个人学习与分享为目的，希望通过造轮子可以提升个人在前端开发中的技能，同时方便更多使用Vue框架的使用者。
